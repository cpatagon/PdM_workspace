Envia una dirección y un comando al dispositivo M\+A\+X7219.

Esta función envía un byte de dirección seguido de un byte de comando utilizando el protocolo S\+PI. La función envía el par de bytes (dirección y comando) múltiples veces según el valor predefinido de \textquotesingle{}num\textquotesingle{}.
