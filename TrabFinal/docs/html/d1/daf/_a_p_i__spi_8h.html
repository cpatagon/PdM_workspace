Envia una dirección y un comando al dispositivo MAX7219.

Esta función envía un byte de dirección seguido de un byte de comando utilizando el protocolo SPI. La función envía el par de bytes (dirección y comando) múltiples veces según el valor predefinido de 'num'.

Parámetros
addressByte de dirección que se desea enviar a través de SPI.
cmdByte de comando que se desea enviar a través de SPI.
