Envía una dirección y un comando utilizando el protocolo SPI manualmente.

Envia una dirección y un comando al dispositivo MAX7219.

Esta función envía un byte de dirección seguido de un byte de comando utilizando el protocolo SPI. La función envía el par de bytes (dirección y comando) múltiples veces según el valor predefinido de 'num' (numero de pantallas led). se recomienda ver el datasheet de CI MAX7219

Parámetros
addressByte de dirección que se desea enviar a través de SPI.
cmdByte de comando que se desea enviar a través de SPI.

Establece el pin CS (Chip Select) en estado bajo para iniciar la comunicación con el dispositivo esclavo conectado puerto cs_PIn mediante el cable CS.

Envía el par de bytes (dirección y comando) 'num' veces en caso de tener multiples pantallas.

Establece el pin CS (Chip Select) en estado bajo indicando indicando que se terminará con la comunicación
